Most of the operations are done in real mode. (16 bit mode).

3.3.1 --> Interrupts
Interrupts are mechanisms that forces the CPU to halt and do some operations. The one
we're using in CHAPTER 3 is the 0x10 (write to display) interrupt, and the 0x13
(I/o interrupt service routine) interrupt are some that exist. 

3.3.2 --> CPU registers
CPU registers have 4 general purpose:
ax, bx, cx, and dx. these are general purpose and can be used for everything.

ALL follow format of:

mov (register), (value)
mov ax, 1234
mov cx, 0x234
mov dx, 't'
mov bx, ax

These registers are 'words' though (2 bytes), so we can divide to the high
and low byte.

mov ax , 0 ; ax -> 0 x0000 , or in binary 0000000000000000
mov ah , 0 x56 ; ax -> 0 x5600
mov al , 0 x23 ; ax -> 0 x5623
mov ah , 0 x16 ; ax -> 0 x1623

    je done			 ; if equal move to done.

    inc si           ; move to next character
    jmp print_loop   ; repeat the print_loop
